MECHENG700 - P4P
Eye-tracking System for Diagnosis of Visual Deficits
Author: S. Raje & J. Zhang

This contains all the MATLAB files used throughout the Part 4 project.
The files include:
	- On screen stimulus visualisation
	- Calibration Test
	- Fixation Tests
	- Trajectory Tests
	- Analyses for each test (Fourier analysis, Gaze position, Calibration, etc.)

The folder also includes a subfolder, "FINAL", of a few subject's results following the tests.

The final report and presentation are shown in the pdf and powerpoint slides respectively.
